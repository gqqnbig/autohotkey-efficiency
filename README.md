# 常见问题
## Chrome浏览器提示文件有毒？

Chrome浏览器（Google Safe browsing）倾向于把AutoHotKey编译的程序报为病毒。如有疑虑，建议使用https://www.virustotal.com 查毒，确保无毒再下载。
